🔧 Configuración Manual de Supabase

Sistema de Encuestas PAE - Envigado

⚠️ Nota: Esta versión no requiere cargar Supabase desde CDN. Te guiará paso a paso para configurar tu base de datos manualmente.

📋 Paso 1: Acceder a Supabase

  1. Ve a: https://supabase.com/dashboard
  2. Inicia sesión con tu cuenta
  3. Selecciona tu proyecto (o crea uno nuevo si no tienes)
  4. Anota la URL y API Key de tu proyecto

🗄️ Paso 2: Ejecutar SQL en Supabase

  1. En el panel de Supabase, ve a "SQL Editor"
  2. Haz clic en "New query"
  3. Copia el SQL que aparece abajo
  4. Pégalo en el editor SQL
  5. Haz clic en "Run" (botón verde)

🧪 Paso 3: Probar la Conexión

Una vez ejecutado el SQL, prueba la conexión:

  1. Abre test-supabase.html en tu navegador
  2. Haz clic en "Probar Conexión"
  3. Deberías ver: "✅ Conexión exitosa!"

✅ Paso 4: Usar la Aplicación

Una vez que la conexión funcione:

  1. Abre index.html para la aplicación principal
  2. O index-v2.html para la versión mejorada
  3. Ve a la sección "Encuestas"
  4. Completa una encuesta de prueba
  5. Verifica que los datos se guarden en Supabase

🚨 Solución de Problemas

Problemas Comunes:

  • Error "relation does not exist": La tabla no se creó. Ejecuta el SQL nuevamente.
  • Error "permission denied": Las políticas RLS no están configuradas. Ejecuta el SQL completo.
  • Error "Failed to fetch": Verifica que la URL y API Key sean correctas.
  • Los datos no aparecen: Verifica que el SQL se ejecutó correctamente.
